# 初めに
x86 16bitモードで`ADC`命令を使用し、32ビットの加算をする検証をします。
検証には8086互換CPUであるV30を搭載したパソコンを使用します。

# 検証用コード及び検証法

<img width="2359" height="1769" alt="df40ffedea34d8" src="https://github.com/user-attachments/assets/417a2bfd-e018-4c60-8737-202ee009230e" />

```ADC.ASM
.model small
.stack 100h
.data
.code
main:
 
 ; num1=65535 dx:ax=0x0000FFFF
 mov dx,0000h
 mov ax,0ffffh
 
 ; num2=1     bx:cx=0x00000001
 mov bx,0000h
 mov cx,0001h

 add ax,cx
 adc dx,bx

 mov ah,4ch
 int 21h
end main
```

```
masm adc.asm
link adc.obj
debug adc.exe
```

`-u`:アセンブリを確認
`-t`:逐次実行
