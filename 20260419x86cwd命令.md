# 初めに

`cwd命令 (Convert Word to Doubleword)`は、AXレジスタの16ビット符号付き整数を、DX:AXレジスタペアの32ビット符号付き整数に変換する。<br>

動作:<br>
    AXの最上位ビット（ビット15） を確認<br>
    もしビット15が0（正の数又は0）なら、DXに0x0000を格納<br>
    もしビット15が1（負の数）なら、DXに0xFFFFを格納<br>


# 実証

```test.asm
section .text
global _start

_start:
    
    mov ax,0b0111111111111111 ; 正の数
    mov dx,0xBBBB
    int3; 初期化後の値を確認

    cwd ; DX = (AXの最上位ビット)0x0000
    int3
    
    mov ax,0b1111111111111111 ; 負の数
    mov dx,0xCCCC
    int3; 初期化後の値を確認

    cwd ; DX = (AXの最上位ビット)0xFFFF
    int3
    
    ; 終了
    mov eax, 1
    mov ebx, 0
    int 0x80
```

```bash
nasm -f elf32 test.asm -o test.o
ld -m elf_i386 test.o -o test
gdb ./test
```

# 追記
8086の互換CPU`V30`でも同じことをやってみた。

```CWD.ASM
.model small
.stack 100h
.data
.code
main:

 mov ax,7fffh  ; 0111111111111111
 mov dx,0bbbbh
 cwd

 mov ax,0ffffh ; 1111111111111111
 mov dx,0cccch
 cwd

 mov ah,4ch
 int 21h
end main
```
