# 環境変数の保存先

環境変数の保存先は以下の2つ

```
コンピューター\HKEY_CURRENT_USER\Environment
コンピューター\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Environment
```

環境変数画面
上段 → ユーザー（HKCU）
下段 → システム（HKLM）
