
2つのレジスタの値を入れ替える

```
masm xchg.asm
link xchg.obj
debug xchg.exe
```

```XCHG.ASM
.model small
.stack 100h
.data
.code
main:
 
 mov ax,1234h
 mov bx,5678h
 xchg ax,bx

 mov ah,4ch
 int 21h
end main
```
