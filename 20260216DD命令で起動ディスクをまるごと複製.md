# 目的

MS-DOSをCFカードから起動しています。
念の為に起動可能なCFカードをもう一つ作って置きます。
起動ディスクはブートセクタ等もありそのままファイルを複製しただけでは起動が出来ないので、
DD命令を使ってまるごと複製します。

# 手順

複製元からimgとして抽出
```
lsblk
sudo dd if=/dev/sdb of=pocket8086.img bs=4M status=progress
```

複製先へimgを書き込む
```
lsblk
sudo dd if=pocket8086.img of=/dev/sdb bs=4M status=progress conv=fsync
```

以下を実行して書き込みが終わって入れば抜きます。
```
sync
```
