
`naked`を付けると以下のようなコードを吐かなくなる。
OS開発では邪魔になることがあるのでこれで消せる
```
push ebp
mov ebp,esp
```

```test.c
void normal_func() {
    asm volatile("nop");
}

__attribute__((naked))
void naked_func() {
    asm volatile("nop");
}
```

```bash
test@test-fujitsu:~/kaihatsu/ctest$ gcc -m32 -O0 -ffreestanding -fno-pie -c test.c -o test.o
test@test-fujitsu:~/kaihatsu/ctest$ objdump -d -M intel test.o

test.o：     文件格式 elf32-i386

   3:	90                   	nop
   4:	90                   	nop
   5:	5d                   	pop    ebp
   6:	c3                   	ret    

00000007 <naked_func>:
   7:	90                   	nop
   8:	90                   	nop
```
