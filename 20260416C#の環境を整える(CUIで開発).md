# 開発環境構築まで

[.NET 10.0 SDK](https://dotnet.microsoft.com/zh-cn/download/dotnet/thank-you/sdk-10.0.202-windows-x64-installer)

```bash
dotnet new console -o AAAA
cd AAAA
```

初期状態では以下のCSファイルが自動生成されている。
```Program.cs
Console.WriteLine("Hello, World!");
```

コンパイル及び実行
```
dotnet run
```

コンパイルのみ
```
dotnet build
```

上記の`Program.cs`を実行すると以下の様に表示される。
```
Hello, World!
```

dllを生成せず単独のEXEを生成したい場合は以下を実行
```
dotnet publish -c Release -r win-x64 --self-contained true /p:PublishSingleFile=true
```

# 検証

## 単一ファイル

```
using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("aaaaaaa");
        Console.WriteLine($"今日は {DateTime.Now:yyyy-MM-dd}");
        Console.ReadKey();
    }
```

```
C:\Users\oe\AAAA>dotnet run
aaaaaaa
今日は 2026-04-15
```

## 複数ファイル

`using`は無くても可

```Program.cs
class Program
{
    static void Main()
    {
        A.ShowA();
        B.ShowB();
    }
}
```

```File1.cs
class A
{
    public static void ShowA()
    {
        System.Console.WriteLine("=== A ===");
    }
}
```

```File2.cs
class B
{
    public static void ShowB()
    {
        System.Console.WriteLine("*** B ***");
    }
}
```

```
C:\Users\oe\AAAA>dotnet run
=== A ===
*** B ***
```
